## ğŸ“œ Historical Context

**XBMC (Xbox Media Center)** began as a media player application for the first-generation Xbox console in 2002. What started as a modded Xbox app evolved into one of the most popular media center platforms, eventually becoming **Kodi** in 2014.

The skinning system was revolutionary for its time, allowing users to completely customize every aspect of their media center interface. This repository preserves these skins as a snapshot of media center UI design from the golden age of home theater PCs.
